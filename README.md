# Jus Clip It

Jus Clip It is an AI-powered SaaS platform that transforms long-form videos into viral short-form clips.

The platform helps creators upload podcasts, interviews, and streams, then automatically detects engaging moments, ranks clips by virality, identifies story arcs, dynamically reframes video into 9:16 vertical format, generates captions, and exports ready-to-post clips for TikTok, Instagram Reels, and YouTube Shorts.

## Core Features

- AI transcription with Whisper
- Story arc detection
- Virality scoring and clip ranking
- Smart vertical reframing similar to Opus Clip
- FFmpeg-based clip rendering
- CapCut-style customizable captions
- Stripe subscriptions and usage tiers
- SaaS dashboard for uploads, clips, and billing
